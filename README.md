# 🍄 Süper Mario: 5. Sınıf Türkçe Serüveni

Bu proje, klasik Super Mario Bros mekaniklerini **5. Sınıf Türkçe** dersi müfredatıyla birleştiren eğitsel bir web oyunudur. Öğrencilerin oyun oynarken dil bilgisi, yazım kuralları ve anlam bilgisi konularını pekiştirmesini amaçlar.

## 🚀 Özellikler
- **Devasa Harita:** 50.000 piksellik prosedürel olarak oluşturulan dev oyun alanı.
- **Eğitici Soru Blokları:** Mario (?) bloklarına vurduğunda karşısına Türkçe soruları çıkar.
- **Doğru Cevap Ödülü:** Soruları doğru bilince Mario devleşir ve ekstra puan kazanır.
- **Hibrit Kontroller:** Hem Klavye (W,A,S,D) hem de mobil dokunmatik desteği mevcuttur.
- **Yakışıklı Mario:** Modernize edilmiş, detaylı karakter tasarımı.

## 🕹️ Nasıl Oynanır?
1. **Hareket:** Klavye yön tuşları veya ekrandaki oklar.
2. **Zıplama:** `Boşluk` tuşu veya `ZIPLA` butonu.
3. **Sorular:** Altın renkli soru bloklarına alttan vurarak soruları tetikleyin.
4. **Devleşme:** Soruyu doğru cevaplayarak 10 saniye boyunca ölümsüz ve dev olun!

## 🛠️ Teknik Detaylar
- **Dil:** HTML5, CSS3, JavaScript (Pure JS)
- **Render:** HTML5 Canvas API
- **Uyumluluk:** Masaüstü ve Mobil tarayıcılar (Responsive)
